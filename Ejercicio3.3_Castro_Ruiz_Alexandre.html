Lista de Definiciones

HTML
HTML (HyperText Markup Language) es el lenguaje de marcado utilizado para estructurar el contenido de las páginas web.
CSS
CSS (Cascading Style Sheets) es un lenguaje utilizado para describir la presentación y el diseño de un documento HTML o XML.
JavaScript
JavaScript es un lenguaje de programación utilizado para crear interactividad en las páginas web, como formularios, animaciones y más.
